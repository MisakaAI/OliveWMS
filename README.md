# OliveWMS

Olive Warehouse Management System
基于 `Django` 开发的简易 WMS 系统

## 功能介绍

- 基础设置
  - 仓库
  - 货架
  - 储位
  - 批次
  - 产品
  - 供应商
- 库存管理
  - 查询
  - 移动
  - 合并
  - 拆分
  - 统计
- 入库
  - 进货
  - 退料
  - 生产入库
  - 销退
- 出库
  - 退货
  - 领料
  - 销售
- 盘点

## 环境部署

- [Python 3.13](https://www.python.org/downloads/)
  - `pip install -i django`
  - `pip install django-cors-headers`
  - `pip install "psycopg[binary]"`
